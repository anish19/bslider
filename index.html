Getting Started

Backbone.Slider (BSlider) is built on top of Backbone.js. Make sure Backbone is setup in your application, and you have the CSS files for Backbone.Slider. Backbone.Slider uses Font Awesome, jQuery-ui, and jQuery. Please be sure to include those.

Backbone.Slider

var MySlider = Backbone.Slider.extend({
    views: [myBackboneViewOne, myBackboneViewTwo] 
}); //The Views should be rendered

var instance = new MySlider().render();

addViews instance.addViews

Add an array of views to an instance of the slider.

Initial View To Be Rendered

When the slider is rendered, if you require a specific view to be shown as the first slide, it can be specified as an option.

Future versions will include selecting initial views by name, id, etc.

var MySlider = Backbone.Slider.extend({
    initialViewToBeRendered: 1,
    views: [myBackboneViewOne, myBackboneViewTwo] 
}); //Index starts at 0

var instance = new MySlider().render();

Crosslinks

Cross-links can be enabled by passing in an option while extending.

var MySlider = Backbone.Slider.extend({
    enableCrossLinks: true,
    views: [myBackboneViewOne, myBackboneViewTwo] 
});

bsliderCrossLinkTitle

Cross-link titles can be specified as a property of the backbone view.

var MyBackboneView = Backbone.View.extend({bsliderCrossLinkTitle: 'My View'}),
    myBackboneViewOne = new MyBackboneView(); //View(s) should be rendered

var MySlider = Backbone.Slider.extend({
    enableCrossLinks: true,
    views: [myBackboneViewOne] 
});

cycleThrough

Enable this option to cycle through views. 

var MySlider = Backbone.Slider.extend({
    cycleThrough: true,
    views: [myBackboneViewOne, myBackboneViewTwo] 
});

preTransition and postTransition

If you want to control what happens before and/or after slide transitions, define preTransition, and/or postTransition functions while extending. 

var MySlider = Backbone.Slider.extend({
    preTransition: function () { //Do something },
    postTransition: function () { //Do something }
});

More features coming soon. Stay tuned!
