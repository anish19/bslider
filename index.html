Backslider

A backbone based slider for backbone views.

Note: This is a very crude version. Since the intent was to release early, a snapshot version has been released

Getting Started

Backbone.Slider (Backslider) is built on top of Backbone.js. Make sure Backbone is setup in your application, and you have the CSS files for Backbone.Slider. Backbone.Slider uses Font Awesome. Please be sure to include that.

Backbone.Slider

var MySlider = Backbone.Slider.extend({
    views: [myBackboneViewOne, myBackboneViewTwo] 
}); //The Views should be rendered

var instance = new MySlider().render();

addView instance.addView

Add one more view, or an array of views to an instance of the slider.

More features coming soon. Stay tuned!
